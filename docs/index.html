My personal-Jukebox

Project Objectives

This project is a small data collection, visualisation and actuation challenge. It integrated hardaware (using a RPI 3B + with and RC522 sensor and NTAG213) and a Deep learning model to predict and recommend music depending on the users clothing style or 'mood'. The ml algorithm uses the songs listened by the user in the past month to make a prediction of what it might want to listen when an item of clothing is scanned. Each item has a selected 'mood' which was found through data analysis from the users trends.

Firstly, below you will see data sets for localised weather from Earls Courts London retrieved via the Open weather API, for the music I listen to every hour on Spotify using the Spotify API and finally a collection of clothing items I have worn in the past weeks using a raspberry pi an RC522 sensor and NTAG213 tags.
