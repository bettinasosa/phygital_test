Project Objectives

This project is a small data collection, visualisation and actuation challenge. It integrated hardaware (using a RPI 3B + with and RC522 sensor and NTAG213) and a Deep learning model to predict and recommend music depending on the users clothing style or 'mood'.

Project Tools

Software

  • Spotipy python library, the Spotify web-API and my Spotify creadentials.
  • Pymongo python library for Mongodb, Mongodb Charts to display data and Mongodb to store the collections.
  • Mfrc522 python library
  • Pandas and Numpy for Data anlasys
  • Keras and Tensorflow to create a Deep learning model to predict my 'music' mood.
  • Sklearn to validate the model.
  • Open weather API

  • RFID-RC522 sensor
